           NPC Memory and Relationship System

           Version 0.1

           1. DESIGN    PHILOSOPHY

           NPCs are not chatbots.

           NPCs are people.

           Every NPC:

              • remembers;
              • forgets;
              • changes opinions;
              • forms relationships;

              • develops fears;
              • creates stories.
           Memory creates personality.

           2. MEMORY     CATEGORIES

           Every NPC possesses several types of memory.

           Personal Memory

           Memories directly involving the NPC.

           Examples:

              • conversations;
              • insults;
              • gifts;
              • promises;

           3. MEMORY     STRUCTURE

           Each memory contains:

              • event;
              • date;
              • location;
              • participants;
              • emotional impact;
              • confidence;
              • importance;
              • source.

           Example:

           Event: "The player saved my daughter."

           Emotion: Gratitude.

           Importance: 95/100.

           Confidence: 100%.

           4. MEMORY     IMPORTANCE

           Importance levels:

           0-20 Minor memory.

           21-50 Normal memory.

           51-80 Important memory.

           81-100 Life-changing memory.

5. MEMORY     DURATION

           Minor memories:

              • days;
              • weeks.

           Normal memories:

              • months.

           Important memories:

              • years.

           Life-changing memories:

              • entire lifetime.

           6. FORGETTING

           NPCs forget naturally.

           Factors:

              • age;
              • intelligence;
              • trauma;
              • repetition;
              • importance.

           Example:

           "Traveler bought bread."

           Forgotten after weeks.

           Example:

           "Traveler killed my brother."

           Never forgotten.

           Every event changes emotions.

           Examples:

              • gratitude.

           Rob NPC:

              • anger.

              • distrust.

           Keep promise:

              • respect.

           -100 Mortal enemy.

           -75 Enemy.

           -50 Disliked.

           0 Neutral.

           25. AI MEMORY      PROCESS

           Before responding, AI checks:

              1. Who is speaking?
              2. Do I remember them?
              3. What emotions do I associate?
              4. Has my opinion changed?
              5. What happened recently?
              6. What rumors have I heard?
