           1. DESIGN    PHILOSOPHY

           Factions are not teams.

           Factions are living organizations.

           Every faction:

              • has goals;
              • has fears;
              • has enemies;
              • has allies;

              • has internal conflicts;
              • changes over time.

           2. FACTION    STRUCTURE

           Every faction consists of:

              • ideology;
              • leadership;
              • members;
              • territory;
              • economy;
              • military;
              • reputation;
              • history.
