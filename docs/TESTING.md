# AI-DND Early Test Checklist

## 1. Test URL

Primary URL:

https://ai-dnd-blue.vercel.app

## 2. What This Test Includes

В первый тест входят:

- открытие игры по ссылке;
- главное меню;
- создание персонажа;
- переход на карту;
- переключение языка RU/EN;
- сохранение в `localStorage`;
- проверка базовых экранов;
- проверка загрузки ассетов.

## 3. What This Test Does Not Include Yet

В этот этап пока не входят:

- настоящий AI-диалог NPC;
- backend;
- аккаунты;
- серверные сохранения;
- общий мир;
- multiplayer/MMORPG;
- синхронизация между устройствами.

Сохранения пока живут только в браузере игрока. Они не отправляются на сервер и не переносятся автоматически на другое устройство или в другой браузер.

## 4. Tester Instructions

1. Открой ссылку.
2. Проверь, что главное меню загрузилось.
3. Нажми **New Game**.
4. Создай персонажа.
5. Дойди до **WorldMap**.
6. Переключи язык RU/EN.
7. Обнови страницу.
8. Закрой и открой игру снова.
9. Проверь, сохранился ли персонаж.
10. Запиши, что сломалось или было непонятно.

## 6. Known Limitations

- AI пока не работает для внешних тестеров.
- Ответы NPC могут быть заглушками.
- Сохранение не переносится между устройствами.
- Очистка данных браузера может удалить сохранение.
- Это ранняя тестовая, а не финальная версия игры.
